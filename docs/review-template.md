## 段落级标注

对每个章节至少给出下面信息：

- `风格`：1-5（个人声音）
- `结构`：1-5（事实到观点推进）
- `可发`：1-5（选题判断力 / 是否值得保留）
- `flags`：从固定标签中选 1-3 个
- `备注`：一句话说清问题

推荐备注句式：

- `这段不该写，因为……`
- `这段只是在复述新闻，没有推进到……`
- `这段结论方向对，但不像我，因为……`
- `如果按我的写法，我会先讲……再落到……`

## 常用 Flags

- 选题类：`选题偏弱` `不适合付费周报` `重要性不足` `偏消费资讯`
- 推进类：`只在复述新闻` `缺少关键背景` `没有推导到影响` `结论跳跃` `判断越界`
- 风格类：`不像我` `机器腔` `太圆滑` `空话偏多` `缺少我的判断`
- 事实类：`来源不够硬` `引用重复` `事实不准` `时间线混乱`

## 训练友好写法

### Chosen Text

贴入你愿意保留或你亲手改好的版本。

### Rejected Text

贴入系统原文，或者你明确不满意的旧版本。

### Reason

优先使用这些短语：

- `topic_selection`
- `insight_upgrade`
- `voice_alignment`
- `market_depth`
- `source_hardening`
- `manual_edit`
