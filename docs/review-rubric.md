# 三维标注标准

这套标注标准服务于当前最关键的三项强化目标：

1. 选题判断力
2. 事实到观点的推进
3. 个人声音

目标不是把每一段都改成完美文章，而是把你的判断过程稳定地沉淀为训练资产，供后续 `PreferencePair`、`SectionScore`、`EditDiff` 和偏好优化使用。

## 一、三大维度

### 1. 选题判断力

关注的不是“这段写得顺不顺”，而是“这个主题值不值得进入你的周报”。

- `5 分`：高度值得写，重要、非共识、和你的长期叙事高度相关
- `4 分`：值得写，但还可以再收束角度
- `3 分`：可写可不写，信息有价值，但不是本周最优先
- `2 分`：偏题、偏浅、偏热闹，放进周报会稀释质量
- `1 分`：不该选，和你的周报定位明显不符

判断时优先问自己四个问题：

- 这件事是否足够重要，而不是只有流量？
- 它是否能承载你的独特解释，而不只是新闻摘要？
- 它是否能和美国经济、美国政治、中国宏观、中国政治、美国科技/市场主线形成连接？
- 它是否适合放进付费内容，而不是免费资讯？

### 2. 事实到观点的推进

关注模型是否真的完成了“从新闻到判断”的跃迁。

- `5 分`：事实、背景、因果、影响、边界都交代清楚，能自然推出你的观点
- `4 分`：已经不只是复述新闻，但结论力度或链条完整性稍弱
- `3 分`：有分析意图，但多数还停留在泛泛影响
- `2 分`：主要是在转述新闻，观点推进很弱
- `1 分`：几乎没有分析，或者推导明显跑偏

理想的推进链条：

1. 发生了什么
2. 为什么值得关注
3. 背后驱动是什么
4. 对政策、市场、行业或资产意味着什么
5. 哪些结论现在还不能下

### 3. 个人声音

关注这段是不是“像你本人”，而不是“像一个会写中文的模型”。

- `5 分`：明显像你，判断语气、节奏、取舍都贴近你的历史文章
- `4 分`：大体像你，只有少量机器腔或废话
- `3 分`：可读，但更像通用财经评论
- `2 分`：机器味明显，空泛、安全、过度圆滑
- `1 分`：完全不像你，甚至出现明显违和的立场或口吻

你个人声音的核心特征，按当前历史语料总结，主要包括：

- 明确下判断，而不是反复兜圈
- 会说“我认为 / 我觉得 / 我的判断是”
- 不满足于复述事实，会继续追问“真正意味着什么”
- 会给出边界感，不会装作什么都确定
- 市场评述强调基本面、估值、预期差、长期主义和投资心态

## 二、后台现有字段怎么映射

当前后台没有为这三项单独建新字段，所以先按下面的映射使用：

- `事实`：用于事实准确性与来源可靠性
- `风格`：用于“个人声音”
- `结构`：用于“事实到观点的推进”
- `可发`：用于“选题判断力 + 本段是否值得留在成稿里”

也就是说，现阶段最关键的三项训练目标，分别主要看：

- `风格 = 个人声音`
- `结构 = 观点推进`
- `可发 = 选题判断力`

## 三、推荐 Flags

建议优先使用固定标签，避免每次发明新说法。

### 选题判断力

- `选题偏弱`
- `不适合付费周报`
- `重要性不足`
- `偏消费资讯`
- `偏热闹不偏关键`

### 事实到观点推进

- `只在复述新闻`
- `缺少关键背景`
- `没有推导到影响`
- `结论跳跃`
- `判断越界`
- `分析深度不足`

### 个人声音

- `不像我`
- `机器腔`
- `太圆滑`
- `空话偏多`
- `缺少我的判断`
- `价值投资味道不足`

### 事实与来源

- `来源不够硬`
- `引用重复`
- `事实不准`
- `时间线混乱`
- `实体识别有误`

## 四、每段备注最好怎么写

备注尽量不要写成“这里不好”。最好按固定句式写，便于后续训练：

- `这段不该写，因为……`
- `这段事实没问题，但没有推到真正重要的结论……`
- `这段结论方向对了，但不像我，主要因为……`
- `如果按我的写法，我会先讲……再落到……`

## 五、最省时间的打标方法

如果你赶时间，每段只做下面四件事就够了：

1. 给 `风格 / 结构 / 可发` 打分
2. 选 `1-3` 个 flags
3. 把你真正会发出去的版本贴进 `Chosen Text`
4. 把原文保留在 `Rejected Text`

这已经足够形成高价值偏好数据。

## 六、判定标准示例

### 示例 A：不该入选

- `可发 1-2`
- flags：`选题偏弱, 偏消费资讯, 不适合付费周报`
- 备注：`这类导购型科技新闻不符合我的周报定位，缺少宏观或投资层面的信息增量。`

### 示例 B：事实在，但观点弱

- `结构 2-3`
- flags：`只在复述新闻, 没有推导到影响`
- 备注：`事实交代了，但没有继续分析这件事对行业格局、监管态度或市场定价意味着什么。`

### 示例 C：内容能用，但不像我

- `风格 2-3`
- flags：`不像我, 机器腔, 缺少我的判断`
- 备注：`文字太安全，缺少明确立场和我一贯的判断力度。`

## 七、使用原则

- 优先标出“最不像你的地方”，不要平均用力
- 优先保留你真正改写后的文本，而不是只打分不改字
- 与其给很多宽泛备注，不如给少量高信噪比修改
- 只要你能稳定打标 `6-10` 个周更周期，模型就会明显往“像你”收敛
